# Парсер магазина 5ka.ru

## Описание
Парсер для сбора данных о товарах с сайта 5ka.ru с сохранением в SQLite базу данных.

## Функциональность
- Парсинг категорий товаров
- Парсинг товаров с полной информацией
- Автоматическая пагинация
- Сохранение в SQLite базу данных
- Анализ изменения цен между запусками
- Экспорт данных в Excel/CSV
